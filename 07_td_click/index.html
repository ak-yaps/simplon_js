TP - Click

Rédiger un petit programme :

  • en HTML, créer un bouton.
  • Utiliser l'attribut onclick pour déclencher une fonction AfficherMessage.
  • Au click sur le bouton, afficher une alerte.
  • En option : encapsuler votre code dans une IIFE
  • Créer plusieurs boutons en HTML.
  • Leur mettre une classe CSS 'btn'.
  • En JS, parcourir tous les .btn avec une boucle.
  • Au cick sur chaque bouton, exécuter la fonction afficher Message

Associer clicks et classes CSS

Rédiger un programme actifInactif :

  • Créer un élément UI (Widget) représentant un utilisateur.
  • Se servir par exemple de font-awesome et son isocne user.
  • Au click, sur l'icône, activer ou desactiver une classe CSS .is-active sur le widget.
  • Si le widget n'est pas actif, ajouter la classe, sinon la retirer.
  • Indice : utiliser une api déjà évoqué et testé en cours.
  • BONUS : ajouter une transition css à l'action
